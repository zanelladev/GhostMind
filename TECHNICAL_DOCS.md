Lógica de Spawn Aleatório\n\n### Função: `chooseRandomGhostSpawns()`\n\nLocalizada em: `lib/game/engine.ts`\n\n**Algoritmo:**\n\n```\n1. Reservar células obrigatórias (Pac-Man, power pills)\n2. Listar todas as células abertas\n3. Filtrar por:\n   - Sem paredes\n   - Sem célula reservada\n   - Distância >= minSpawnDistance de Pac-Man (Manhattan)\n4. Selecionar aleatoriamente ghostCount células\n5. Fallback: usar GHOST_SPAWNS clássicos se insuficientes\n```\n\n**Complexidade:**\n- Espaço: O(rows × cols) para listar células\n- Tempo: O(maxSpawnAttempts × candidates.length)\n- Prático: < 5ms em maze 21x21\n\n### Função: `buildGhosts()`\n\nConstrói array de GhostState baseado em settings:\n\n```typescript\nfunction buildGhosts(\n  settings: GameSettings,\n  walls: boolean[][],\n  pacmanPos: Vec,\n): GhostState[] {\n  const count = Math.min(Math.max(settings.ghostCount, 1), GHOST_SPAWNS.length);\n  const spawnCells = settings.ghostSpawnRandom\n    ? chooseRandomGhostSpawns(walls, pacmanPos, settings)\n    : GHOST_SPAWNS.slice(0, count);\n\n  return spawnCells.map((pos, index) => ({\n    id: `ghost-${index}`,\n    personality: PERSONALITIES[index % PERSONALITIES.length],\n    pos: { ...pos },\n    home: { ...pos },\n    metrics: { nodesExpanded: 0, pathLength: 0, timeMs: 0 },\n    frightened: false,\n  }));\n}\n```\n\n## 3.
