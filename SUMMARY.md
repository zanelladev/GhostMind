original\n- Sem surpresas\n\n### Modo Aleatório (ON)\n- Cada jogo gera posições aleatórias\n- Validação automática (sem paredes, distância mínima)\n- Fallback inteligente se não conseguir posições\n- Respawn mantém randomização\n\n### Configuração Dinâmica\n- Mude settings antes de iniciar\n- Clique \"Reiniciar\" para aplicar\n- Cada parâmetro é independente\n- Salvo por sessão (não persiste entre recargas)\n\n## 🎯 Casos de Uso\n\n### Tutorial\n- Spawn OFF, 1 fantasma, Fácil\n- Ideal para aprender mecânicas\n\n### Casual\n- Spawn ON, Distância 8, 1-2 fantasmas, Fácil\n- Divertido e acessível\n\n### Competitivo\n- Spawn ON, Distância 6, 3 fantasmas, Hard\n- Máximo desafio\n\n### Clássico\n- Spawn OFF, 3 fantasmas, Normal\n- Experiência original\n\n## 📈 Roadmap Futuro\n\n- [ ]
